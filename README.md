## 回復者数が常に0で表示される
現在利用しているAPI内のデータプロバイダーが回復者数を提供していないようです  
そのため現状アプリでは感染者数と死亡者数のみ閲覧可能です

## データ
コミュニティによりOSSで開発されているAPIを利用します
